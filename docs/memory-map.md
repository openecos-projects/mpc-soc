# 地址空间

当前内存映射同时体现在 `config/memory.yml`、`hw/include/soc_pkg.sv` 和 `sw/ecos/board.h` 中。

| 区域 | 基地址 | 大小 | 说明 |
| --- | ---: | ---: | --- |
| flash | `0x3000_0000` | `0x0100_0000` | 外部 SPI flash XIP 窗口 |
| psram | `0xc000_0000` | `0x0180_0000` | 主要软件 MEM 区域，3 片 8MiB |
| clint | `0x0201_0000` | `0x0001_0000` | Core-local 中断控制器 |
| plic | `0x0c00_0000` | `0x0040_0000` | Platform-level 中断控制器 |
| uart0 | `0x1000_0000` | `0x0000_1000` | 控制台 UART 占位 |
| rcu | `0x1000_2000` | `0x0000_1000` | 复位和时钟控制单元 |
| rtc | `0x1000_4000` | `0x0000_1000` | 实时时钟 |
| wdg | `0x1000_5000` | `0x0000_1000` | 看门狗定时器 |
| archinfo | `0x1000_6000` | `0x0000_1000` | 架构信息寄存器 |
| gpio0 | `0x1010_0000` | `0x0000_1000` | GPIO 控制器 |
| uart1 | `0x1010_3000` | `0x0000_1000` | APB4 UART 控制器 |
| i2c | `0x1010_4000` | `0x0000_1000` | APB4 I2C 控制器 |
| pwm | `0x1010_6000` | `0x0000_1000` | PWM 控制器 |
| timer0 | `0x1010_8000` | `0x0000_1000` | APB 定时器 |
| timer1 | `0x1010_9000` | `0x0000_1000` | APB 定时器 |
| timer2 | `0x1010_a000` | `0x0000_1000` | APB 定时器 |
| timer3 | `0x1010_b000` | `0x0000_1000` | APB 定时器 |
| qspi | `0x1020_0000` | `0x0000_1000` | APB4 QSPI 控制器 |
| rng | `0x1030_0000` | `0x0000_1000` | 伪随机数生成器 |
| crc | `0x1030_1000` | `0x0000_1000` | CRC 计算引擎 |

在生成器覆盖全部目标之前，需要保持硬件 package、软件头文件和文档同步。
