# mpc-soc

`mpc-soc` 是一个整理中的 SoC 参考工程目录，当前主要包含已有构建产物、外设 IP 与新增 IP 相关代码：

- `build/`：当前生成或整理出的顶层相关文件。
- `perip/`：已有外设模块代码。
- `new-ip/`：新增或待集成的 IP 模块、测试与文档。

## 当前状态

本目录目前不是正式完成的 SoC 模板，也不是最终生产版本。代码和目录结构仍处于整理、验证和集成参考阶段，可能存在接口未统一、文档不完整、测试覆盖不足或实现细节待调整等情况。

请将本工程仅作为参考材料使用，用于理解模块组织、外设实现和后续 SoC 集成方向。不要直接将其作为稳定模板、课程最终提交模板或生产环境交付版本使用。

## 使用建议

- 在复用任何模块前，请先检查对应 IP 的接口、时钟复位、总线协议和寄存器定义。
- 在集成到新的 SoC 顶层前，请补充必要的仿真、综合和板级验证。
- 如需形成正式模板，应进一步统一目录规范、构建脚本、文档说明和回归测试流程。
